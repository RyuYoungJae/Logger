# Logger

## ConsloeLog

### 요구사항
* 로그 레벨은 INFO, ERROR가 존재, [I],[E]로 표기
* 로그 양식은 [로그 남은 날짜][로그 레벨][내용]
* % 값을 기준으로 인자를 대입 하여야한다.
* 로그는 콘솔, 파일에 남아야 한다.
* 윈도우 콘솔에는 INFO는 하얀색, Error는 빨간색으로 노출.
* 파일은 특정 사이즈를 초과 시, 특정 시간마다 새로 만들어져야 한다.
